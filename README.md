# CV Creation Website with AI Enhancement

## Overview
This repository contains the source code and documentation for a CV creation website that incorporates artificial intelligence to enhance user experience. The website allows users to create professional CVs more efficiently and effectively.

## Features
- **AI Suggestions**: Automatically suggests improvements to CV content based on industry standards.
- **Template Selection**: Offers a variety of professionally designed templates to choose from.
- **Real-time Preview**: Users can see how their CV will look in real-time as they edit.
- **Export Options**: Ability to download the finalized CV in various formats (PDF, DOCX).

## Getting Started
To get started with the project, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/truongson04/SS2_Final_Demo.git
   cd SS2_Final_Demo
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the application**:
   ```bash
   npm start
   ```

## Usage
- Create a new account or log in.
- Start creating your CV by filling out the necessary information.
- Use the AI suggestions to improve your CV content.
- Choose a template and customize it according to your preferences.
- Finally, export your CV to your desired format.

## Technologies Used
- HTML/CSS for the frontend
- JavaScript for the interactivity
- Node.js for the backend
- Various AI libraries for suggestion generation
